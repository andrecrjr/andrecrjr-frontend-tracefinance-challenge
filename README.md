# Andrecrjr Frontend Tracefinance Challenge

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

You can run the tests in:
```bash
npm run test
# or
yarn test
# or
pnpm test
# or
bun test
```

Environment Variables:
INTERN_API="https://fe-challenge-trace-api-production.up.railway.app"
